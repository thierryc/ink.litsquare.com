# LitSquare Ink MD (Website)

Static website for the LitSquare Ink MD landing page.

## Local preview

Run a static server from the repo root:

```sh
python3 -m http.server 8000
```

Then open `http://127.0.0.1:8000/`.

## Notes

- Icons live in `src/` and are referenced by `index.html`.
- The typing animation uses `typed.js` loaded from a CDN.
